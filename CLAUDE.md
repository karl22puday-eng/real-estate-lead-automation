## Prime directives (non-negotiable)
1. **Correctness over confidence.** No invented n8n node names, params, or API fields. If
   unsure how a node behaves in the installed version, verify or ask.
2. **Every external call handles failure** — timeout, non-2xx, empty/malformed payload,
   rate limit. Retries on the HTTP/Telegram nodes.
3. **Idempotency by design.** Re-runs / duplicate webhooks never create duplicate leads or
   double alerts. Dedup short-circuit + upsert on a natural key.
4. **Secrets never hardcoded.** Keys live in n8n credentials / gitignored `.env`. Only
   `.env.example` is committed.
5. **Validate at the edge.** The webhook validates + normalizes before any downstream node.
6. **Reproducible infra.** Workflows are generated by the `scripts/*.ps1` from source, not
   hand-edited JSON. Edit the script, regenerate, re-import.
7. **Test before declaring done.** Trace hot, cold, duplicate, and bad-input paths; report
   what I actually ran.
8. **Truthful status.** Untested / partial / placeholder is stated plainly.
